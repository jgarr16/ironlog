Open the app Live site

North star

  • Mobile-first gym tracker for two lifters — fast to log, easy to read side-by-side.
  • Single-file PWA (index.html), no build step; data stays in the browser (localStorage).
  • Machine reference photos in-repo so each exercise can show the right station at a glance.

Where we are now

  • Five training days + Notes tab; John / Kyong weights on one card per exercise.
  • Bundled thumbnails for many exercises from the images/ folder (chest, shoulders, legs, cable stack, pallof, etc.).
  • Triceps extension shares the cable-station photo with lat pulldown (same machine); optional custom image URL per exercise still available (tap thumbnail).
  • Weekly export to ironlog-history.json on “Start New Week.”

Where we’re going next

  • Photos: Add a standing triceps pushdown shot (different from seated extension / cable stack). Fill in assets for exercises that still use the img placeholder (e.g. seated row, face pull, pec fly, calf raise on leg press) as you get pictures.
  • Polish: Keep layout tight on small phones as more thumbnails and labels land.
  • README alignment: Update docs if the shipped feature set drifts from README.md (program tables, Pallof naming, etc.).
